# Library Management System - Mini Project

This is a **mini library management system** designed for testing purposes.  
It allows to manage users, books, and loans with basic functionalities.

## Features

- User registration
- Add books to the library
- Borrow books
- Return books
- Track borrowed books and due dates
- Ensure availability and proper tracking of all loans

## Tech Stack

- Node.js + Express for backend
- Jest + Supertest for functional and API testing
- Mock in-memory database for simplicity

## How to Run

1. Install dependencies:

```bash
npm install
```
2. Run Tests

``` bash
npm test
```
